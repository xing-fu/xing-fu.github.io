## 快捷键

- P - 暂停 / 继续
- O - 设置

## 彩蛋

- 点击左上角暂停图标暂停 / 继续动画效果。
- 点击上方扬声器图标开启 / 关闭音效。
- 点击右上角齿轮图标进行个性化设置。
- 拖动屏幕底部调整动画速度。
